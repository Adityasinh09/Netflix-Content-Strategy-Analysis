# 🎬 Netflix Content Strategy Analysis

## 📌 Project Overview
An end-to-end data analysis project exploring Netflix's global content library to uncover trends in media formats, genre popularity, regional production, and historical growth. This project demonstrates the full data pipeline: data cleaning and transformation in Python, followed by interactive data visualization in Tableau to extract actionable business insights.

## 🛠️ Tech Stack
* **Data Cleaning & Wrangling:** Python (Pandas), Jupyter Notebook
* **Data Visualization & Dashboarding:** Tableau Public
* **Dataset:** Netflix Movies and TV Shows Dataset

## 💡 Key Insights Discovered
Based on the visual analysis, several key strategies regarding Netflix's content library emerge:
1. **Format Dominance:** The platform's library is heavily skewed toward Movies rather than TV Shows.
2. **Explosive Growth:** There was an exponential surge in content added to the platform starting around 2014, peaking heavily between 2018 and 2019.
3. **Top Performing Genres:** "Dramas, International Movies" and "Documentaries" lead the platform in total volume, indicating a strong focus on global, real-world, and dramatic storytelling.
4. **Global Production Hubs:** The United States produces the vast majority of the content library (over 2,800 titles), followed distantly by India and the United Kingdom.
